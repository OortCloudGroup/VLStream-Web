# 算法仓库模块

算法仓库模块负责管理和展示各种AI算法，提供算法的查看、分类、搜索等功能。

## 模块结构

```
AlgorithmWarehouse/
├── AlgorithmManagement.vue    # 算法管理主页面
├── index.js                   # 模块统一导出
└── README.md                  # 模块说明文档
```

## 功能特性

### AlgorithmManagement.vue - 算法管理
- **分类标签**: 支持按算法类型筛选（全部、人员检测类、视频分析类、人脸识别类）
- **算法网格**: 以卡片形式展示算法列表
- **响应式设计**: 支持不同屏幕尺寸的自适应布局
- **交互效果**: 卡片悬浮动画、渐入动画等

## 设计特点

1. **简洁的分类导航**: 顶部标签页形式的分类切换
2. **现代化卡片设计**: 包含算法图片、名称和分类信息
3. **流畅的交互体验**: 悬浮效果、动画过渡
4. **响应式布局**: 适配移动端和桌面端

## 技术实现

- 使用 Vue 3 Composition API
- CSS Grid 实现响应式网格布局
- CSS 动画实现交互效果
- 组件化设计，易于扩展和维护
