# 图片存储目录说明

## 概述
这个目录用于存储算法标注页面上传的图片文件。

## 功能特性

### 1. 图片上传
- 用户在算法标注页面上传图片时，图片会自动保存到此目录
- 支持的图片格式：jpg, jpeg, png, gif, bmp, webp
- 文件大小限制：最大10MB
- 文件名格式：`时间戳_原始文件名`

### 2. 图片存储
- 所有上传的图片都存储在 `VLStream-ui/src/img/` 目录下
- 图片文件名使用时间戳前缀避免重复
- 在数据库中保存图片的相对路径：`/src/img/文件名`

### 3. 图片加载
- 从数据库加载标注数据时，系统会自动从此目录加载对应的图片
- 如果图片文件不存在，系统会提示用户需要重新上传图片

## 技术实现

### 后端API
- **上传接口**：`POST /api/image/upload`
- **删除接口**：`DELETE /api/image/delete`
- **检查接口**：`GET /api/image/exists`

### 前端功能
- 图片拖拽上传
- 自动保存到本地目录
- 支持多图片管理
- 图片预览和缩略图

## 使用说明

1. **上传图片**：
   - 在标注页面点击上传区域或拖拽图片文件
   - 系统自动保存到本地目录并在数据库中记录路径

2. **标注图片**：
   - 选择标注工具（矩形/圆形）
   - 在图片上绘制标注框
   - 右键选择标签类型
   - 点击"保存当前标注"按钮

3. **查看标注**：
   - 重新进入标注页面
   - 系统自动加载保存的图片和标注数据
   - 标注框会正确显示在图片上

## 注意事项

- 确保VLStream-server后端服务已启动
- 图片上传需要后端API支持
- 删除标注项目时，相关的图片文件不会自动删除
- 建议定期清理未使用的图片文件

## 故障排除

### 图片无法显示
1. 检查图片文件是否存在于此目录
2. 检查数据库中imagePath字段是否正确
3. 确认后端服务是否正常运行

### 图片上传失败
1. 检查文件格式和大小限制
2. 确认后端API接口是否可访问
3. 检查目录权限是否允许写入
